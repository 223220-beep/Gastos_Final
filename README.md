# Gastos App (Mobile)

Una aplicación Android moderna para la gestión de gastos personales, desarrollada íntegramente en **Kotlin**.

## Características
- **Gestión de Gastos**: Registra y visualiza tus gastos diarios de forma rápida.
- **Recordatorios**: Configura recordatorios para pagos o gastos futuros.
- **Análisis**: Visualización gráfica de gastos por categoría y mes (usando MPAndroidChart).
- **Inteligencia de Voz**: Integra comandos de voz para registrar gastos y recordatorios rápidamente.
- **Persistencia Local**: Utiliza Room para un almacenamiento seguro y eficiente de los datos.

## Tecnologías Utilizadas
- **Lenguaje**: Kotlin
- **Arquitectura**: MVVM (Model-View-ViewModel)
- **Base de Datos**: Room Persistence Library
- **Inyección de Dependencias**: Hilt (Dagger)
- **Asincronía**: Coroutines & Flow
- **UI**: XML con ViewBinding
- **Gráficos**: MPAndroidChart

## Cómo Ejecutar el Proyecto
- Abrir el proyecto directamente en **Android Studio**.
- Dejar que Android Studio sincronice los archivos de **Gradle**.
- Ejecutar en un emulador o dispositivo físico con Android 7.0 (API 24) o superior.
